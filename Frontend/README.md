# Portfolio Frontend

React + Vite portfolio application with Tailwind CSS.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Update the API URL in `.env` file

4. Run development server:
```bash
npm run dev
```

5. Build for production:
```bash
npm run build
```

## Features

- Home page with hero section
- About page with skills and resume download
- Projects showcase with filtering
- Blog section
- Contact form
- Admin dashboard for managing projects
- Authentication system
- Responsive design with Tailwind CSS

## Tech Stack

- React 18
- Vite
- React Router v6
- Axios
- Tailwind CSS
